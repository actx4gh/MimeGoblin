# Privacy

MimeGoblin does not send data anywhere.

## What it reads
The extension reads download metadata that Chrome provides to extensions via the Downloads API (filename, URL, referrer, MIME type, etc.). It does not read page content.

## What it stores
- Your routing rules and settings are stored using Chrome's storage APIs.
- An optional local routing log is stored so you can see what matched recently.

## Network access
The extension does not make network requests and does not load remote code.

## Permissions
- downloads: needed to apply routing suggestions
- storage: needed to save rules and settings
